# 欢迎来到我的技术博客！

今天成功搭建了自己的技术博客，使用的是 GitHub Pages + Jekyll + Chirpy 主题。

## 为什么要写博客？

1. **记录学习过程** - 把学到的知识整理成文章，加深理解
2. **分享技术经验** - 帮助遇到同样问题的人

4. **提升表达能力** - 通过写作提高技术表达能力

## 博客搭建过程

### 选择方案

经过对比，我选择了 GitHub Pages + Jekyll + Chirpy 主题：

- ✅ 完全免费
- ✅ 无需服务器
- ✅ 使用 Git 管理
- ✅ Markdown 写作
- ✅ 主题美观

### 搭建步骤

1. 从 Chirpy Starter 模板创建仓库
2. 克隆到本地
3. 配置 `_config.yml`
4. 写第一篇文章
5. 推送到 GitHub
